Instructions

It is suggested to use google chrome on a PC desktop. Following the link first...

Please install these extensions for google chrome:

TransOver
Then:

  • click Add to Chrome
  • Click puzzle-like icon on the top right of chrome
  • Locate TransOver on the drop list, click vertical '...', then Options
  • (Maybe) select 'Hungarian' 'when I pointed at word'
  • then 'save' then 'more options'
  • (Maybe) select 'from English' then check and select 'Enable text to speech when I press ctrl'
  • then 'save' and abandon the options page (close tab)
  • So from now on, if you points to any word or selection, a translation pops up

Dictionary Anywhere
Then:

  • click Add to Chrome
  • Click puzzle-like icon on the top right of chrome
  • Locate Dictionary Anywhere on the drop list, click vertical '...', then Options
  • 'English' and trigger key:'Alt'
  • then 'save' and close it
  • So from now on, if you take "Alt+double clicks" on any word or selection, a small thesaurus pops up



Mobile Alternatives. Please install these apps:

Google Translate on Android
Then:

  • By default, Google Translate exposes itself to the context menu of another applications, so a Google Translate command will be showing in dropdown menus when you double-tap over a text (thus making a selection).

Google Translate on Iphone or any
Then:
  • Try to circumvent choosing any application option, which let you allow your Translate app to expose itself to the context menu of another applications (or so). Then, the Translate command will appear in dropdown menu (or so) when you somehow make a text selection.

Speech synthetizers (English should be preselected in both). Dropdowns should be populated in any uncorrupted up-to-date modern browser installation. Please select your favourite ones (2 different ones, for example a US-english and a UK-english). If lists are empty please fix your browser (preferable google chrome) installation or consult the software provider.
NOTE! It doesn't work well in Android, where the system's default synthetizer is the only one operational, regardless to website configuration.
